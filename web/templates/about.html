关于美股量化分析系统

美股量化分析系统是一个基于 Python 的专业股票分析工具，提供技术指标分析、基本面分析、新闻情绪分析和市场扫描功能。

核心功能

  • ✓ 技术分析：RSI、MACD、布林带、SMA 50/200
  • ✓ 基本面分析：PE、PB、ROE、营收增长
  • ✓ 新闻情绪分析：基于 TextBlob 的 NLP 分析
  • ✓ 市场扫描：自定义条件筛选投资机会
  • ✓ 专业主题：深色/浅色金融风格界面

技术栈

后端

  • Flask - Web 框架
  • yfinance - Yahoo Finance 数据
  • finvizfinance - 筛选和新闻
  • ta - 技术分析库
  • TextBlob - 情绪分析

前端

  • 原生 HTML/CSS/JavaScript
  • ECharts - 专业图表库
  • CSS Variables - 主题系统
  • Responsive Design - 响应式布局

数据来源

  • 价格数据：Yahoo Finance (yfinance)
  • 财务数据：Finviz
  • 新闻数据：Yahoo Finance News
